# Sutton Tech LLC - Corporate Website

Welcome to the official repository for the Sutton Tech LLC corporate website framework.

## Overview
This repository contains the optimized frontend codebase built to host Sutton Tech's three primary online presences seamlessly.

### File Structure
- `index.html`: The primary corporate homepage highlighting the business infrastructure and Upper Peninsula roots.
- `business-connect.html`: The definitive landing platform for the **Business Connect** SaaS PSA product suite.
- `dgx-spark.html`: A highly optimized product page for the **DGX Spark** enterprise GPU compute rentals.
- `styles.css`: The centralized global stylesheet for unified brand identity.
- `scripts.js`: Consolidated vanilla JS logic for frontend interactions.

## Tech Stack
Engineered for raw speed, security, and instantaneous GitHub Pages deployment.
- Strict HTML5 Semantic architecture
- **Tailwind CSS** (via CDN)
- **Vanilla JavaScript** (Zero-Framework overhead)

## Local Development
Because the application is driven entirely by vanilla web technologies with no build steps, you can start local development instantly:
1. Clone this repository.
2. Open `index.html` in any web browser.
3. Or launch via any modern lightweight server (e.g. VSCode Live Server).
